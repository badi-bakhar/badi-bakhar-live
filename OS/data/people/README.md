# data/people/

Notes about people — thinkers, creators, writers, researchers, and contacts worth tracking.

One file per person.

## File Naming Convention

```
YYYY-MM-DD-firstname-lastname.md
```

Example: `2026-03-17-naval-ravikant.md`

Use the date you first created the note, not any other date. The person's name forms the slug.

## Frontmatter

```yaml
---
title:
date: YYYY-MM-DD
source: web | youtube | instagram | manual
role:
tags: []
---
```

- `role` is a short description: `investor`, `author`, `economist`, `poet`, etc.
- `source` is where you first encountered this person.

## Suggested Structure

```markdown
## Who they are
Brief bio in your own words.

## Key ideas
- Idea 1
- Idea 2

## Where to find their work
Links to books, channels, accounts, etc.

## Notes
Anything else — personal impressions, connections to other notes.
```

## Notes

This directory is for people who appear across multiple contexts or whose ideas you want to track over time. Not every person mentioned in a note needs their own file here.
